### Description
Arranges windows in a Fibonacci spiral or dwindle.

### Scheme
```
+-----------+-----------+  +-----------+-----------+
|           |           |  |           |           |
|           |     2     |  |           |     2     |
|           |           |  |           |           |
|     1     +--+--+-----+  |     1     +-----+-----+
|           | 5|-.|     |  |           |     |  4  |
|           +--+--+  3  |  |           |  3  +--+--+
|           |  4  |     |  |           |     | 5|-.|
+-----------+-----+-----+  +-----------+-----+-----+
	 spiral                     dwindle
```
