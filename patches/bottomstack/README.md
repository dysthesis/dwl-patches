### Description
bstack and bstackhoriz are two stack layouts for dwl.
### Scheme
```
bstack        (TTT)       bstackhoriz   (===)
+-----------------+       +-----------------+
|                 |       |                 |
|                 |       |                 |
|                 |       |                 |
+-----+-----+-----+       +-----------------+
|     |     |     |       +-----------------+
|     |     |     |       +-----------------+
+-----+-----+-----+       +-----------------+
```
